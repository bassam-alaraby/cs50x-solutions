# Week 0 — Scratch

**Topic**: Visual programming, computational thinking

## Problems

### Colourful Dino 🦕
A side-scrolling game built in Scratch where a dinosaur jumps over cacti.
- Score increases as you survive longer
- Game over screen on collision
- Custom sprites drawn in Arabic (الديناصور، الصبار، الخلفية)
