# Week 1 — C

**Topic**: C basics, conditionals, loops, operators

## Problems

### Hello (me & world)
Two versions of a simple hello program — one greets the user by name, one prints "Hello, World".

### Mario (less & more)
Prints a pyramid of `#` blocks.
- **Less**: single pyramid (right-aligned)
- **More**: double pyramid with a gap in the middle
- Takes height as input, validates 1–8

### Cash
Calculates minimum number of coins for change using a greedy algorithm.
Coins used: 25¢, 10¢, 5¢, 1¢
