# Week 9 — Flask

**Topic**: Backend web development, Python, SQL, sessions

## Problems

### Birthdays
A CRUD web app to store and display friends' birthdays.
Flask + SQLite — supports adding and deleting entries via a form.

### Finance
A stock trading simulator.
- Register/login with hashed passwords
- Look up real stock prices via IEX API
- Buy and sell shares, tracked in SQLite
- Portfolio view with current values and P&L
- Full transaction history
- Extra features: change password, add cash
