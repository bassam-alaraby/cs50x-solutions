# Week 2 — Arrays

**Topic**: Arrays, strings, command-line arguments, cryptography basics

## Problems

### Scrabble
Compares two words by their Scrabble point values and declares a winner.
Uses a predefined points array mapped to each letter.

### Readability
Calculates the Coleman-Liau index of a text to estimate reading grade level.
Counts letters, words, and sentences to compute the formula.

### Caesar
Encrypts a message using Caesar cipher with a key passed as a command-line argument.
Wraps around the alphabet and preserves case.
