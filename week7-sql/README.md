# Week 7 — SQL

**Topic**: Relational databases, SQL queries, joins, aggregation

## Problems

### Songs
SQL queries on a Spotify database of songs and artists.
Covered: SELECT, WHERE, JOIN, ORDER BY, AVG, subqueries.

### Movies
SQL queries on an IMDb database of movies, people, and ratings.
More complex joins across multiple tables (movies ↔ stars ↔ people ↔ ratings).

### Fiftyville
A mystery puzzle solved entirely with SQL.
Queried crime scene reports, interviews, flight records, and phone logs
to identify the thief, their escape city, and their accomplice.
Full query log saved in log.sql, answers in answers.txt.
