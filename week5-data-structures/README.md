# Week 5 — Data Structures

**Topic**: Linked lists, hash tables, tries, dynamic memory

## Problems

### Inheritance
Simulates genetic inheritance of blood types across three generations.
Uses a linked-list-style family tree allocated with malloc and freed recursively.

### Speller
Spell-checker that loads a dictionary into a hash table and checks a text file.
- Hash function distributes words across 1000+ buckets
- Average O(1) lookup per word
- Reports misspelled words and performance benchmarks
